# OmniScribe PWA Icons

The manifest at `/manifest.json` references:

- `/icons/icon-192.png` — 192×192 PNG (any purpose)
- `/icons/icon-512.png` — 512×512 PNG (any + maskable)

These are generated by **`scripts/generate-pwa-icons.mjs`** — a dependency-free
raw-PNG encoder. Run it to (re)produce all PWA + favicon assets:

```sh
node scripts/generate-pwa-icons.mjs
```

That writes `public/icons/icon-192.png`, `public/icons/icon-512.png`,
`src/app/icon.png` (the Next App Router favicon), and `public/favicon.ico`.

The current mark is a white leaf on the brand teal (`#3d8b8b`) — the same
motif as the `BrandWordmark` quill, kept inside the maskable safe area. The
binaries are committed; re-run the script and commit the output whenever the
brand mark changes. Swap in designer-produced artwork for production by
replacing the files (the manifest paths stay the same).
